

Option C -- Default Case.  This configuration option determines whether 
ECHO defaults to sending text in upper-case or lower-case.  The default can 
be changed within the text string with the "%<" and "%>" escape sequences.


Option D -- Default Destination.  Whether ECHO defaults to console or 
printer output of the text string is set by this option.  The selected 
default can be changed within the text string with the "%C" and "%P" escape 
sequences.
