
Option Q -- Quiet mode.  This option determines whether DATSTP will be in 
verbose or quiet mode by default.  The default chosen here may be toggled 
by using the "Q" option on the command line.

Option T -- Time format.  Select either civilian (12-hour) time or military 
(24-hour) time for the screen display.  Command line entry is always 24- 
hour time.

Option D -- Date format.  Select either American or European date order for 
both display and command line entry.  The American order is "mm/dd/yy".  
The European order is "dd/mm/yy".
